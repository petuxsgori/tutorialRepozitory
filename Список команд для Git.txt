Создание репозитория: git init

Клонирование репозитория с облака: git clone ссылка-на-репозиторий

Добавление определенного файла в репозиторий: git add "Список команд для Git".txt

Добавление всех файлов в папке: git add .

создание версии: git commit -m "init"

загрузка изменений:git commit -am "Что было сделано3"

откат изменений:git commit --amend -am "Что было сделано3"

Переход на другой путь(обычная консольная команда): cd C:\Users\ayuanisimov\Desktop\папка git

История изменений коммита: git diff ID-коммита

История коммитов: git log

Список веток: git branch

Создание ветки: git branch имя-ветки

Переход на ветку: git checkout имя-ветки

Переход и создание ветки: git checkout -b имя-ветки

Подключение к репозиторию на облаке: git remote add origin [ссылка на репозиторий]

Добавление изменений на облако: push origin master
